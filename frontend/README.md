# Vue Landing Page

This project is a simple Vue.js landing page that features a navbar and a list of categories.

## Project Structure

- `src/assets/styles/main.css`: Contains the main styles for the application.
- `src/components/NavBar.vue`: Defines the NavBar component with a "Home" link.
- `src/components/CategoryList.vue`: Displays a list of categories.
- `src/views/Home.vue`: Main view that uses NavBar and CategoryList components.
- `src/App.vue`: Root component of the application.
- `src/main.js`: Entry point of the application.
- `public/index.html`: Main HTML template for the application.
- `package.json`: Configuration file for npm.
- `vite.config.js`: Configuration for Vite.

## Installation

1. Clone the repository.
2. Navigate to the project directory.
3. Run `npm install` to install dependencies.
4. Run `npm run dev` to start the development server.

## Usage

Open your browser and go to `http://localhost:3000` to view the landing page.
